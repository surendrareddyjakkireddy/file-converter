**Core responsibility**:



Accept DOCX

Convert to PDF

Return downloadable file



**Functionality**

The system allows users to upload DOCX files and convert them into PDF format using a Spring Boot-based REST API. Through this project, I learned how to handle file uploads, process document conversion, and return downloadable files. It also helped me understand frontend-backend integration and RESTful service design.
